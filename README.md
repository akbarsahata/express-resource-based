# express-resource-based
a personal experiment to restructure an Express project by not using MVC

## How to use (in local)
modify `.env.example` to `.env` and replace `dbUrl` on app.js. Finally, run this command

> npm run dev

## How to use (in production)

> sudo su
> npm run serve
